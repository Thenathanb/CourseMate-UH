# CourseMate

**Instantly see professor ratings while browsing supported university course catalogs (currently University of Houston)!**

## What it does

CourseMate automatically displays RateMyProfessors ratings directly next to instructor names on supported university course search pages (currently UH). No more switching between tabs or searching manually - see ratings, difficulty, and student reviews at a glance.

### Features

- **Instant Ratings** - Color-coded ratings appear automatically (green for great, yellow for good, red for challenging)
- **Detailed Info** - Hover to see rating count, difficulty, "would take again" %, grade distributions, and recent reviews
- **Quick Access** - Click "View Profile" to read full RateMyProfessors reviews
- **Smart Caching** - Ratings are cached for 7 days to ensure fast performance
- **Currently UH-Only** - Only shows University of Houston (Main Campus) professors

### How to Use

1. Install the extension
2. Navigate to any supported course search page (currently *.uh.edu)
3. Ratings automatically appear next to professor names
4. Hover over ratings for more details
5. Click "View Profile" to see full reviews on RateMyProfessors.com

### Privacy

This extension:
- Only works on supported university websites (currently *.uh.edu)
- Fetches data directly from RateMyProfessors public API
- Stores professor ratings locally in your browser for faster loading
- Does NOT collect, store, or transmit any personal information
- Does NOT track your browsing history

### Support

Found a professor that isn't showing ratings? Some professors may not have reviews on RateMyProfessors yet. The extension only displays ratings for professors with existing reviews.

Having issues? Please report bugs or request features on our GitHub page.
