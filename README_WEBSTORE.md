# CourseMate UH - RateMyProfessors for UH

**Instantly see professor ratings while browsing University of Houston course catalogs!**

## What it does

CourseMate UH automatically displays RateMyProfessors ratings directly next to instructor names on UH course search pages. No more switching between tabs or searching manually - see ratings, difficulty, and student reviews at a glance.

### How to Use

1. Install the extension
2. Navigate to any UH course search page (*.uh.edu)
3. Ratings automatically appear next to professor names
4. Hover over ratings for more details
5. Click "View Profile" to see full reviews on RateMyProfessors.com
